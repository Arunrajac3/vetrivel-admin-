📄 PDF Import — தமிழ் PDF இருந்து வினாக்கள்

PDF upload பண்ணவும் → உரை extract ஆகும் → கீழே "வினாக்களாக பிரி" பட்டனை அழுத்தி ஒவ்வொரு வினாவையும் தனித்தனி பெட்டிகளில் எளிதாக edit பண்ணலாம் → சரிபார்த்து Firestore-க்கு upload பண்ணலாம்.
Pattern: 1. கேள்வி... A) ... B) ... C) ... D) ... Answer: B
